# arcor2_object_types

Object type base classes plus the bundled “built-in” action object types (flow, logic, time, random).

## Usage

- Import object type helpers from `arcor2_object_types` (e.g., `abstract.py`, `utils.py`, test object definitions).
- Upload the built-in types with the `arcor2_upload_object_types` PEX/Docker image.

## Versioning

- `VERSION` contains the current package version.
